# Pi-Zero-Stealth-HID-Proxy
一個可以讓你的MX Master系列滑鼠在公司也能發揮100%功能的小裝置

## 你需要準備以下設備
- Raspberry Pi Zero 2W一台
- micro SD卡一張（8GB以上）
- USB Type Micro-B to USB Type-A線2條
- USB Type-A插頭1個
